# Assessments Formais

## Objetivo

Concentrar pareceres formais datados usados para registrar cortes executivos de maturidade, calibracoes de baseline e decisoes de `go/no-go`.

Esta pasta existe para diferenciar:

- documentos vivos e canonicos na raiz de `docs/`
- artefatos historicos de apoio em `docs/history/`
- pareceres formais datados em `docs/assessments/`

## Quando Usar

Use esta pasta quando a necessidade for:

- congelar a leitura consolidada de um corte especifico
- registrar um parecer executivo formal
- sustentar reunioes de calibracao de baseline e `go/no-go`

Nao use esta pasta como fonte primaria para:

- leitura curta de sponsors
- explicacao viva da baseline
- board diario de execucao
