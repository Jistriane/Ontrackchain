## Regra de Precedencia para Janela Seria Integrada

- use `SERIOUS_WINDOW_FINAL_EXECUTION_PACKET.md` como fonte canônica unica para a execucao integrada `P0-01 + P0-02 + P0-03`
- os guias individuais `P0-01`, `P0-02` e `P0-03` continuam como apoio especializado por trilha
- `FIRST_COMBINED_*` funciona como apoio operacional preenchivel para a primeira janela combinada, nao como substituto do pacote final canônico
- `P0-04_REGULATORY_BUNDLE_EXECUTION_GUIDE.md` detalha apenas a trilha especializada do bundle regulatorio oficial
