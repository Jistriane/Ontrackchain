# Templates de Governanca

## Objetivo

Concentrar os modelos reutilizaveis do rito semanal, da janela seria e da execucao por evidencia.

Use esta pasta para:

- criar novos registros semanais
- abrir novos ciclos `D1-D7`
- iniciar war rooms e tracking ao vivo

Nao use esta pasta para:

- guardar registros fechados
- anexar saidas geradas automaticamente
- publicar artefatos datados de uma janela ja executada
